Data Science Classes

Foundations of Data Science

The UC Berkeley Foundations of Data Science course combines three perspectives: inferential thinking, computational thinking, and real-world relevance. It teaches critical concepts in computer programming and statistical inference alongside hands-on analysis of real-world datasets.

The Structure and Interpretation of Computer Programs

An introduction to programming and computer science focused on abstraction techniques to manage program complexity. Topics include functional, object-oriented, and declarative programming, with significant projects.

Data Structures

Fundamental dynamic data structures, including linear lists, queues, trees, and other linked structures; arrays strings, and hash tables. Storage management. Elementary principles of software engineering. Abstract data types. Algorithms for sorting and searching. Introduction to the Java programming language.

Economic Models

Designed to bridge the gap between data science and economics by using real-world datasets and Python programming to illustrate key economic concepts. The course is intended for students from both data science and economics backgrounds, giving them a chance to apply computational tools to economic models, exploring economic concepts such as supply and demand, market equilibrium, utility, game theory, and more. The course emphasizes how economic decisions are influenced by real-world data and policy interventions, making it a practical intersection of economics and data science.

Econ 2

Provides an overview of the field of economics. It covers both microeconomics, the study of consumer choice, firm behavior, and market interaction, and macroeconomics, the study of economic growth, unemployment, and inflation.

Discrete Mathematics and Probability Theory

Logic, infinity, and induction; applications include undecidability and stable marriage problem. Modular arithmetic and GCDs; applications include primality testing and cryptography. Polynomials; examples include error correcting codes and interpolation. Probability including sample spaces, independence, random variables, law of large numbers; examples include load balancing, existence arguments, Bayesian inference.

Sonification of Go Games with Space-filling Curves

Data sonification translates data into sound. This project sonified Go games with space-filling curves, turning a match between AlphaGo and Lee Sedol into music. Using modified Peano curves, Go board coordinates were mapped to 1D space, creating musical translations.
